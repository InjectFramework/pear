InjectFramework PEAR channel

Registering the channel: 
pear channel-discover injectframework.github.com/pear
Listing available packages: 
pear remote-list -c injectfw
Installing a package: 
pear install injectfw/package_name
Installing a specific version/stability: 
pear install injectfw/package_name-1.0.0
pear install injectfw/package_name-beta
Receiving updates via a feed: 
http://injectframework.github.com/pear/feed.xml
